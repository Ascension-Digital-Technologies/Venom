# Venom v1.30.8

Adds browser-side script execution selection.

- Default: scripts compile to protected QuickJS/WASM bytecode.
- File directive: `// @venom: browser` in the leading comment block.
- HTML directive: `<script data-venom="browser" ...>`.
- Browser-selected scripts are retained as package source and executed sequentially in the native browser realm.
- Browser scripts do not use or enable the protected host-JS fallback path.
- The basic-site chess example marks jQuery, chessboard.js, chess.js, Bootstrap, and main.js as one browser-realm dependency chain.
